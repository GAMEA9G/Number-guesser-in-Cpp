My First C++ Project

Experience

I am relatively new to C++, so it was overwhelming at first.

Installation

It was really easy. Just run this command: sudo pacman -S gcc

Hard Part

I wanted IntelliSense, so I needed to install the C/C++ extension from Microsoft. However, I couldn't find it in the marketplace and searched a lot.

Using C++

Running C++ was the hard part because I didn't know the syntax yet, so many errors popped up.

Running C++

It was easy to run the code once I figured it out. To compile and create an executable, use the command: g++ filename.cpp -o executable_name. Then, run the executable with: ./executable_name
